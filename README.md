## How to Edit This Code

You can edit and contribute to this project in multiple ways:

### Use Your Preferred IDE

Work locally using your favorite code editor. Requirements: Node.js & npm installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

Steps:

```sh
# Step 1: Clone the repository
git clone https://github.com/kiranvasala24/AI-Tools.git

# Step 2: Navigate to the project directory
cd ai-productivity-hub

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
